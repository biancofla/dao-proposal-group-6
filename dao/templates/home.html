Read Carefully

The following is a mock application that simulates the behavior of a simple Decentralized Autonomous Organization on the Algorand testnet blockchain. Within this application, users can create polls, cast their votes, and see the poll's final results.

DAOMock is a proof-of-concept application created as the final project of the International School on Algorand Smart Contract. It was created for demonstration purposes only and should never be used in a production environment.

Create a Poll

Create a poll on the Algorand testnet blockchain.

See Existing Polls

Click on one of the existing polls to see its details. If the poll is currently in progress, you can cast your vote. Otherwise, you can see the poll's final results.
